# 🛡️ Bunker: Web Version

**Bunker** is a survival **web game** based on the original **Bunker (Pygame)** project.  
This is a browser adaptation that preserves the core mechanics while introducing new features and modern UI.

## 🌐 About the Project

In **Bunker Web Version**, you play as a survivor trying to outlast an apocalyptic event while staying inside a bunker.  
Manage your resources, monitor your health, and make difficult decisions to survive.

### 🎥 Inspired by Lololoshka's Story LP

This game is **fan-made** and inspired by the *Bunker* series from **Lololoshka's story-based YouTube Let's Plays**.  
It is **not intended for commercial use** and is created purely for educational and entertainment purposes.

## ⚙️ Features

- 🌍 **Web-based interface** (Flask backend + HTML/CSS frontend)
- 🥫 **Resource management** (food, water, health, suit charge, etc.)
- 🗂️ **Inventory system**
- 📜 **Quests and events**
- 💾 **Save & Load system (JSON)**
- 🗣️ **Dialogs and localization** (currently English & Ukrainian)
- 🔧 **Debug menu (for testing & development)**

## 🚧 Current Status

This project is in **Alpha**.  
Some mechanics are still being ported from the original Pygame version, and new web-specific features are under active development.

## 🖥️ Technologies Used

- **Python (Flask)** – Backend & routing
- **HTML/CSS/JS** – Frontend
- **JSON** – Save system & game data
- **Jinja2** – Templates for the UI

## 🚀 How to Run

1. Install dependencies:

```bash
pip install flask
````

2. Run the server:

```bash
python app.py
```

3. Open your browser and go to:

```
http://127.0.0.1:5000/
```
